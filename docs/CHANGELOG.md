# Changelog

## [1.0.0] - 2026-05-09

### Added
- Sistema de autenticación con roles (admin, mesero, cliente)
- Módulo de gestión de menú con CRUD de productos
- Módulo de pedidos con estados (pendiente, preparación, entregado)
- Vista de cocina para gestión de órdenes en tiempo real
- Módulo de reservas con validación de disponibilidad
- Módulo de facturación con cálculo automático de IVA
- Módulo de reportes con gráficos de ventas
- Dashboard administrativo con estadísticas
- Datos de semilla para pruebas
- Arquitectura SPA con enrutamiento por hash
- Persistencia de datos en LocalStorage (frontend)
- API REST completa con Express y MySQL
